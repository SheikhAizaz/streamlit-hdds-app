# streamlit-hdds-app

The deployed web app is live at https://hdds-streamlit.herokuapp.com/
This web app predicts the heart disease as a function of their input parameters (age,sex,cp,rbs,chol,fbs,restecg,thalach,exang,oldpeak,slope,ca,thal).

The web app was built in Python using the following libraries:

streamlit
pandas
numpy
scikit-learn
sklearn
pillow
pickle
